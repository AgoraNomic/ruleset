The Agoran Rulesets

Welcome. If you are trying to access the ruleset of Agora Nomic, you've come to the right place. This website is maintained by Trigon, the current Rulekeepor of Agora Nomic.

The Agoran Rulesets come in two flavors: The Short Logical Ruleset (SLR) and the Full Logical Ruleset (FLR).

  • The Short Logical Ruleset contains the full text of all currently enacted rules, organized into categories. If you are new to Agora, this is the most accessible version.
  • The Full Logical Ruleset contains the same rules as the SLR in the same order, along with a list of changes that rule has gone through as well as some other notable historical annotations. Note that the one linked above is usually going to be unofficial and updated to include the most recent changes. The most recent published FLR is found here.

Additionally, others have made software to track the ruleset. Here's a list of links to those. Warning: these all navigate off-site.

  • Gaelan's ruleset viewer (source) displays the SLR in HTML format with a table of contents. This website is generated from current ruleset data like the textual rulesets linked above.
  • Gaelan's old ruleset code does the same thing as the code I use to generate the rulesets, except it was written in Ruby.
